# User stories Android application
________________

*	Als gebruiker wil ik een melding ontvangen of ik mijn bluetooth wil inschakelen wanneer mijn bluetooth niet ingeschakeld is, waarna de bluetooth ingeschakeld wordt.
*	Als gebruiker wil ik een bericht kunnen ontvangen via BLE.
*	Als gebruiker wil ik een bericht kunnen versturen via BLE.
*	Als gebruiker wil ik een ontvangen bericht kunnen doorsturen via BLE.
*	Als gebruiker wil ik een melding ontvangen of ik mijn wifi wil inschakelen wanneer mijn wifi niet ingeschakeld is, waarna de wifi ingeschakeld wordt.
*	Als gebruiker wil ik een bericht kunnen ontvangen via UDP.
*	Als gebruiker wil ik een bericht kunnen versturen via UDP.
*	Als gebruiker wil ik een ontvangen bericht kunnen doorsturen via UDP.
*	Het systeem moet een broadcast address moeten kunnen initializeren voor de verbinding met UDP.
*	Het systeem moet een interface bevatten voor alle relevante methodes voor zowel BLE als UDP.
*	Het systeem moet berichten over UDP kunnen encrypten en decrypten.
*	Het systeem moet berichten over BLE kunnen encrypten en decrypten.
*	Het systeem moet zijn vector klok kunnen ophalen en vergelijken met de vector klok van ontvangen berichten.
*	Het systeem moet goed getest worden met Unit testen.
